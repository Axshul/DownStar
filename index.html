Installation

Getting started with DownStar is incredibly simple. Just include the module you want to use via a script tag:

HTML
<script src="https://Axshul.github.io/DownStar/[module-name]/Call.js"></script>

Replace [module-name] with the module you want to use (e.g., toast, modal, utils).

Available Modules

DownStar offers a growing collection of useful modules:

Toast Notifications

Beautiful, customizable toast notifications for user feedback with multiple styles and positions.

Usage Examples

Here's a complete example using the Toast module:

example.html
<!DOCTYPE html>
<html>
<head>
  <script src="https://Axshul.github.io/DownStar/toast/Call.js"></script>
</head>
<body>
  <button onclick="DownStar.toast.show('Success!', 'success')">
    Show Toast
  </button>
</body>
</html>

Visit the GitHub repository for more detailed examples and API documentation.
